# WhatsApp Automation Bot

Projeto de automação de envio de mensagens no WhatsApp Web utilizando Python.

## Tecnologias
- Python
- Selenium
- Tkinter
- WebDriver Manager

## Funcionalidades
- Selecionar arquivo TXT
- Enviar mensagens automáticas
- Delay configurável
- Loop infinito
- Interface gráfica

## Como executar

pip install -r requirements.txt

python whatsapp-message-automation.py
